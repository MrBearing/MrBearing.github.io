# Description
大阪でロボット屋をしています。元々設備屋でFA・産ロボの機械設計をしてました。独立->廃業でサラリーマンしてます。
ROS/ROS 2のロボット関連のソフトウエア開発をメインに、たまに架台やエンドエフェクタの機械設計したりしてます。
